Road Safety Analysis in Buenos Aires

Comprehensive study of traffic accidents using data analysis and visualization techniques

Project Description

This project analyzes traffic accident data in Buenos Aires City to identify patterns, risk factors and propose preventive measures. The analysis includes interactive visualizations and a complete Power BI dashboard for decision-making.

Methodology

1. Data Collection

  • Official data sources (City Government)
  • Open data APIs
  • Web scraping of public reports
  • Historical weather data

2. Processing

  • Data cleaning and standardization
  • Address geocoding
  • Accident type classification
  • Creation of derived variables

3. Analysis

  • Exploratory data analysis (EDA)
  • Identification of temporal patterns
  • Spatial analysis (hotspots)
  • Predictive risk modeling

4. Visualization

  • Interactive heat maps
  • Time series
  • Correlation charts
  • Power BI dashboard
