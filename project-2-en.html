Multi-brand RAG Chatbot

Enterprise conversational AI assistance platform

Developer: Vera Guill√©n | Version: 1.0.1 | Main Stack: Python, FastAPI, PostgreSQL, Docker, Azure, RAG

Video Demo

Multi-brand Chatbot is an enterprise intelligent chatbot platform that integrates multiple cutting-edge technologies to provide automated conversational assistance through WhatsApp. The system uses Retrieval-Augmented Generation (RAG) with vector search to provide accurate contextual responses based on business documentation.

Highlighted Features

  • Complete Asynchronous Architecture with FastAPI and SQLAlchemy async
  • Advanced RAG System with PGVector for semantic search
  • Multi-Platform Integration (WhatsApp Business API, Chainlit web interface)
  • Conversational State Machine for complex flows
  • Containerized Deployment with Docker and Azure App Service
  • Observability and Resilience with circuit breakers, Redis cache and structured logging

Containerization, Deployment and Security

Multi-Stage Dockerfile

Optimized Dockerfile with a 'builder' stage for compiling dependencies and a lightweight 'runtime' stage with a non-root user for enhanced security.

Startup Script (startup.sh)

Ensures database and Redis availability, runs Alembic migrations and starts Gunicorn with configurable workers.

Security Measures

  • Secrets Management: Environment variables, no hardcoding
  • SSL/TLS: Forced connections in production
  • Validation: Pydantic for all data input
  • Compliance: Endpoints for Privacy Policy, Terms of Service and Data Deletion
